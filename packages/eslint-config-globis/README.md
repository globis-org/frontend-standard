# eslint-config-globis

[![npm version](https://badge.fury.io/js/eslint-config-globis.svg)](https://badge.fury.io/js/eslint-config-globis)

## Install

```
yarn add -D eslint-config-globis \
            eslint \
            prettier \
            typescript
```

## Usage

Add "extends": "globis" to your .eslintrc
